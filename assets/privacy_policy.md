## 导言

_MK-BOT_ 是一款运行在 Android 上的 **Lua 声明式 UI 运行时**：应用界面、导航与大部分功能由 **Lua 脚本**在本地声明式定义，可在不重新编译 APK 的情况下通过脚本更新行为与界面。

本应用可在设备内提供 **Ubuntu 容器环境（proot）**、**多标签终端**、**内嵌 WebView**，以及可选的 **NapCat / 咔咔珂** 等集成入口。上述可选能力需要您主动安装、配置并联网下载第三方资源，**不属于本应用默认必选功能**。

我们**不运营**用于识别个人身份的后台数据库，也**不要求注册账户**。仅在以下**必要且最小化**场景涉及信息：

## 3. 权限使用说明

权限仅在相关功能需要时申请或使用：

| 权限 | 用途 |
|------|------|
| **INTERNET / ACCESS_NETWORK_STATE** | Lua 脚本与容器内网络请求、WebView、可选框架下载安装 |
| **存储与媒体访问**（含 MANAGE_EXTERNAL_STORAGE、READ/WRITE_EXTERNAL_STORAGE、READ_MEDIA_*） | 读写 Ubuntu 容器文件、Lua 脚本访问共享存储、备份与恢复 |
| **POST_NOTIFICATIONS** | 前台服务/任务状态等必要提醒（可按系统设置关闭） |
| **FOREGROUND_SERVICE**（含 DATA_SYNC 等类型） | 保持容器相关后台任务与 Love 画布服务运行 |
| **REQUEST_IGNORE_BATTERY_OPTIMIZATIONS** | 可选，用于降低后台被系统杀进程的概率（需您授权） |
| **RECORD_AUDIO** | Love2D / 沙盒音频相关能力在需要时申请；默认不主动使用 |
| **WAKE_LOCK** | 部分后台任务保持唤醒 |

本应用**不使用**相机、精确位置、蓝牙、电话等权限。Manifest 中若存在由依赖库引入但未实际调用的权限声明，以**实际代码行为**为准。

您可在系统设置中随时撤回权限；撤回后相关功能可能不可用，但不影响不依赖该权限的其他功能。
